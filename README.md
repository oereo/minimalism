## Commit rule
```console
git commit -m "text"
```

|    Keyword    | Description                    |
| :-----------: | :----------------------------- |
|    [feat]     | 코드나 테스트, 예제, 문서 등의 추가가 있을 때 사용 |
|  [refactor]   | 올바르지 않은 로직을 고친 경우에 사용          |
|    [style]    | 코드에 대한 스타일을 바꿀 때               |
| [docs(guide)] | 문서 작업을 할 때                     |
